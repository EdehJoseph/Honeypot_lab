# Cowrie Honeypot Lab

## Overview

This project demonstrates the deployment of a Cowrie SSH honeypot on Kali Linux running in VMware Fusion. The honeypot captures SSH login attempts and records attacker activity for analysis.

## Technologies Used

- Kali Linux
- Cowrie Honeypot
- Python
- Matplotlib
- VMware Fusion
- Git
- GitHub
